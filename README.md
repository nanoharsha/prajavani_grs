# Prajavani GRS
Open-source Decentralised Grievance Redressal System for Indian State Governments.
Built on Frappe Framework. Supports Mandal → District → State escalation hierarchy,
ATR workflows, appeals, and SMS notifications.

## Install
```
bench get-app https://github.com/your-org/prajavani_grs
bench --site yoursite install-app prajavani_grs
bench --site yoursite migrate
```
